# pdf-chatbot

Developed an AI-powered PDF chatbot using Streamlit, FAISS, SentenceTransformers, and OpenRouter APIs to perform semantic question-answering over uploaded documents using Retrieval-Augmented Generation (RAG).

Demo URL: https://pdf-chatbot-royyashi.streamlit.app/
